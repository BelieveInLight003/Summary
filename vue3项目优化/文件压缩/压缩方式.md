### 压缩方式对比

| 维度       | gzip         | Brotli（br）                  |
| ---------- | ------------ | ----------------------------- |
| 额外压缩率 | 0%（基准）   | 比 gzip 再小 **20-30%**       |
| 压缩耗时   | 快           | 慢 1.5-2×（可离线一次性完成） |
| 解压耗时   | 相近         | 相近                          |
| 浏览器兼容 | IE6+         | Chrome 50+/Edge/Firefox 44+   |
| 服务器支持 | 100%         | Nginx≥1.11.5、Apache≥2.4.26   |
| 推荐用途   | 动态实时压缩 | 静态资源离线压缩              |

### 动静态结合压缩方式

- vite.config.ts 通过 vite-plugin-compression 插件构建，两种类型的压缩包
- 服务器，nginx 配置，动静态资源 根据动静类型不同，择优配置
